# Aidora – Smart Assistance Platform

## Overview
Aidora is a web platform designed to connect families with caregivers and volunteers to assist elderly people and patients.

This project was developed as part of the PIDEV – 3rd Year Engineering Program at Esprit School of Engineering (Academic Year 2025–2026).

## Features
- User Management
- Profile Management
- Intelligent Matching
- Emergency Assistance Mode

## Tech Stack

### Frontend
- Twig
- Bootstrap

### Backend
- Symfony
- PHP
- MySQL

## Architecture
MVC architecture using Symfony Framework.
